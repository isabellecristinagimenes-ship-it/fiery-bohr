# Real Estate Sheets API

Backend para leitura de métricas imobiliárias a partir do Google Sheets.

## Setup Rápido

1. Instalar dependências: `npm install`
2. Configurar `.env` (ver `.env.example`)
3. Rodar: `npm run dev`

## Deploy

Este projeto está pronto para o **Railway**.
Configure as variáveis de ambiente `GOOGLE_SHEETS_ID`, `GOOGLE_SERVICE_ACCOUNT_EMAIL` e `GOOGLE_PRIVATE_KEY` no painel do Railway.
